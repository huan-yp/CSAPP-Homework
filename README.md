## Env Prepare

### WSL

我用的 WSL.

```
wsl --install
```

直接安装即可, 安装后找到 "Windows" 功能, 打开 "适用于 Linux 的 Windows 子系统", "虚拟机平台" 即可.

重启后 `wsl` 进入 Linux 环境.
